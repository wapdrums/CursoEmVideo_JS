1 - Qual empresa criou o JavaScript?

R: O JavaScript foi criado pela Netscape Communications Corporation em 1995. Brendan Eich foi o desenvolvedor principal da linguagem. Atualmente, o JavaScript é mantido pela Ecma International, uma organização sem fins lucrativos que desenvolve padrões para várias tecnologias, incluindo JavaScript.

R: Sim, a ECMAScript é a especificação padrão em que a linguagem JavaScript é baseada. Ou seja, o JavaScript é uma implementação da especificação ECMAScript.

  • string: representa um texto;
  • number: representa um valor numérico, tanto inteiro quanto decimal;
  • boolean: representa um valor verdadeiro ou falso;
  • undefined: representa um valor que não foi definido;
  • null: representa um valor nulo ou vazio;
  • symbol: é um valor único e imutável usado como chave de propriedade de objeto.

    • 4 - o que significa colocar o valor null dentro de uma variavel?

    Em JavaScript, definir uma variável com o valor null significa que ela não possui nenhum valor ou conteúdo definido, sendo uma forma explícita de indicar que a variável existe, mas não possui nenhum valor ou objeto válido até que seja atribuído um valor posteriormente. É importante lembrar que null é diferente de undefined, que é usado para indicar que uma variável foi declarada, mas ainda não possui um valor atribuído.
