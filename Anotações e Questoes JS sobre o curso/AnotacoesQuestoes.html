1 - Qual a diferença entre um cliente e um servidor?

R: Um cliente é um programa ou dispositivo que pede coisas para outro programa ou dispositivo, chamado servidor. O servidor é responsável por fornecer as informações solicitadas pelo cliente. É como um cliente em um restaurante que faz um pedido e o servidor (garçom) traz a comida.

R: A maioria dos sites populares na internet usa JavaScript, mas alguns exemplos incluem: Google, Facebook, Amazon, Twitter, YouTube, LinkedIn, Instagram, Netflix, entre outros.

Modulo A - Conhecendo o javascript

1 - Qual empresa criou o JavaScript?

R: O JavaScript foi criado pela Netscape Communications Corporation em 1995. Brendan Eich foi o desenvolvedor principal da linguagem. Atualmente, o JavaScript é mantido pela Ecma International, uma organização sem fins lucrativos que desenvolve padrões para várias tecnologias, incluindo JavaScript.

2 - Java e Javascript possuem alguma relacao?

Apesar do nome semelhante, Java e JavaScript não têm relação direta. São duas linguagens de programação diferentes, com propósitos e características distintas.

3 - A ecmaScript tem a ver com JavaScript?

R: Sim, a ECMAScript é a especificação padrão em que a linguagem JavaScript é baseada. Ou seja, o JavaScript é uma implementação da especificação ECMAScript.

4 - Softwares criados em javascript?

R: Facebook, google maps, spotify, trello, discord entre outros
